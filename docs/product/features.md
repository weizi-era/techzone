# TechZone - 数码电子产品电商首页

## 产品概述
TechZone 是一个数码电子产品电商平台首页，面向科技爱好者与数码消费者，提供沉浸式的产品浏览与选购体验。主打手机、电脑、耳机、智能穿戴等 3C 数码品类。

## 核心功能

### 1. 轮播图 Hero 区域
- 自动轮播主打新品与促销活动
- 支持手动切换与指示器导航
- 每张轮播图包含标题、描述、CTA 按钮
- 背景采用渐变色调，突出科技感

### 2. 商品分类导航
- 横向分类卡片：手机、电脑、耳机、智能穿戴、摄影器材、游戏外设
- 每个分类配有图标与名称
- 悬停动效展示

### 3. 热门商品展示
- 商品卡片网格展示（4列）
- 每张卡片包含：商品图、名称、价格、标签（热销/新品/折扣）
- 悬停抬升效果
- 支持快速加入购物车

### 4. 促销活动区域
- 限时折扣倒计时
- 爆款专区横向滚动展示
- 促销标签与原价/折扣价对比

### 5. 页面框架
- 顶部导航栏：Logo、搜索框、购物车图标
- 底部 Footer：关于我们、联系方式、社交媒体

## 页面结构
```
Header (导航栏)
├── Logo
├── 搜索框
├── 购物车 / 用户图标

Hero (轮播图)
├── Slide 1: 新品发布
├── Slide 2: 限时折扣
├── Slide 3: 会员专享

Categories (商品分类)
├── 手机
├── 电脑
├── 耳机
├── 智能穿戴
├── 摄影器材
├── 游戏外设

HotProducts (热门商品)
├── 商品卡片 x 8

Promotion (促销活动)
├── 限时折扣 (倒计时)
├── 爆款专区 (横向滚动)

Footer (页脚)
├── 关于我们
├── 客户服务
├── 联系方式
```

## 数据模型
商品数据为前端 Mock，无需后端 API。

## API 端点
无（纯前端展示页面）
